### 3-1. D8 친퀘테레→베네치아 장거리열차 (La Spezia→Venezia S.Lucia)
- **최적안 근거**: 직통 없음, Milano Centrale 단일 환승이 시간·요금 모두 우위. 권장안 **06:51→11:34 (4h43, 환승2회), 2인 €84.20** = 147,771원(오전도착 최속). 최저안 06:06→12:20 (6h14), 2인 €61.70 = 108,284원이나 정오 후반 도착으로 1박 일정 손해. Milano→Venezia Frecciarossa 매시 운행이라 환승 리스크 낮고, 통합권(동일 운송사 1매) 발권 시 환승 실패해도 무료 재배정.
- **예약 방법·링크**: Trenitalia 공식 [trenitalia.com/en.html](https://www.trenitalia.com/en.html) 에서 La Spezia Centrale→Venezia S.Lucia 통합 검색 → Economy/Super Economy 최저가 확보. 대안 Italo [italotreno.com](https://www.italotreno.com/en).
- **예약 시점(예약창 오픈)**: 출발 약 4개월 전 = **2026-07월 말 전후**. 6/12월 시각표 개정으로 일시 30일로 축소 가능성 있어 재확인. **오픈 즉시 예약** 권장(최저가 빠르게 소진).
- **사전예약 이득**: **강력 권장 (예)**. 조기예약 시 Base 대비 대폭 할인 + 정원제라 인기편 매진 가능. 환불정책: Super Economy(변경·환불 불가) + tiRimborso €2 옵션 시 출발 2일 전까지 10% 공제 환불.

### 3-2. D7 친퀘테레 Cinque Terre Treno MS Card + Via dell'Amore
- **최적안 근거**: 친퀘테레선 무제한 + 블루트레일 + Via dell'Amore 접근을 묶은 카드. 9/22(화)는 Band C(주말 12·13·19·20·26·27) 아님 → **Band B €29.50/인 가정** (2인 €59, Band A €22 가능성 있어 재확인). Via dell'Amore는 카드 필수(단독 입장 불가) + 30분당 200명 정원제라 성수기 매진 위험 → 슬롯 확보 위해 카드도 사전 구매 권장.
- **예약 방법·링크**: [card.parconazionale5terre.it](https://card.parconazionale5terre.it) / [trenitalia.com cinque-terre-ms-card](https://www.trenitalia.com/en/services/cinque-terre-ms-card.html). Via dell'Amore 슬롯: [viadellamore.info/en](https://www.viadellamore.info/en). 카드 구매 시 'Via dell'Amore 버전' 선택 또는 슬롯 추가(슈퍼먼트 약 €10 발생 가능, 재확인).
- **예약 시점**: 카드 자체는 가격 변동 없으나 **Via dell'Amore 슬롯 확보 목적상 방문 며칠 전 사전 구매 권장**. 개인 예약창 정확한 오픈일(방문 N일 전)은 viadellamore.info에서 재확인 → **예약창대기** 성격. (단체 최대30명은 72h 전)
